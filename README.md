# Kullanım :

XAMPP'i başlatmak için ./xampp.sh -b yada ./xampp.sh --basla
<br>
XAMPP'i durdurmak için ./xampp.sh -d yada ./xampp.sh --durdur
<br>
XAMPP'i tekrar başlatmak için ./xampp.sh -r yada ./xampp.sh --restart
